# Anubix Tech Club Landing Page

## Project Idea
This is a responsive landing page for the "Anubix Tech Club," built to showcase upcoming workshops and tech events.

## Project Structure
The project is built using React and TypeScript, bootstrapped with Vite. All reusable UI elements are neatly organized inside the `src/components/` directory to maintain clean and scalable code. 

## Components Used
1. **Navbar**: A responsive top navigation bar.
2. **Hero**: A welcoming banner with a call-to-action button featuring a hover effect.
3. **Card**: A reusable TypeScript component used dynamically to display upcoming events.
4. **Footer**: A simple bottom bar.

## How to Run
1. Clone this repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local development server.
4. Open the `localhost` link in your browser.
